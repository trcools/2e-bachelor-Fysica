# Cursus

Deze map dient om cursusmateriaal voor de vakken te bewaren: notebooks, slides, opdrachten en datasets.

Structuursuggestie:

- `HXX-NaamHoofdstuk/` → per hoofdstuk voorbeelden en oefeningen
- `slides/` → presentaties
- `notebooks/` → Jupyter notebooks
- `assignments/` → oefenopdrachten en oplossingen

Voeg hier gerust bestanden toe of maak submappen aan volgens bovenstaande structuur.
